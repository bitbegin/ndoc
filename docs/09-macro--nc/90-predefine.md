## predefined variable

### #line/#column

```
print-line [#line "x" #column]
```

### #version

```
print-line #version
```

### #burc

```
print-line #burc  ;-- yes
```

### #size?

```
#size? "string"    ;-- 7
```

### #align?

```
print-line #align? ;-- 4 or 8
```
