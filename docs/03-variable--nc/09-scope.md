## scope

```
do [
  declare integer! a: 0
]

while [

][
  declare
]

if  xx [
  declare
]

forever [
  declare
]

loop count [
  declare
] 

```

### can't use these words as local varialbe

* `declare`
* `const`
* `=`
* `func`
* `function`

### can't use these words as function arguments

* `declare`
* `const`
* `=`
* `func`
* `function`
* `->`
