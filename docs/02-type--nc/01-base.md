## integers

### signed integers

* i8!
* i16!
* i32!
* i64!
* i128!

```
-1i8
-2i16
-3i32
-4i64
-5i128
```

### unsigned integers

* u8!
* u16!
* u32!
* u64!
* u128!

```
1u8
2u16
3u32
4u64
5u128
```

### size integer

* isize!/int!
* usize!/uint!

```
;-- default(isize!/int!)
-123

;-- isize!/int!
-123I

;-- usize!/uint!
456U
```

### hex format

```
0x1234

0x1234u8
0x1234u16

```
